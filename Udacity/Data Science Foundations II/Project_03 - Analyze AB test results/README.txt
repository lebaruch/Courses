# General Information

In this project, I had to understand the results of an A / B test run by an e-commerce site. Its purpose is to work with this notebook to help the company understand whether to implement the new page, maintain the old page, or run the test longer before making the decision.

# Version and libraries

- Python 3.6
- Pandas
- Numpy
- Matplotlib

# Files

- `ab_data`: database used to gather information
- `countries`: database that has more information than the original, such as Countries
- `Analyze_ab_test_results_notebook`: notebook with all the project and HTML version for better view
